# Real-Time Object Detection and Tracking using AI and Webcam

This project is a Streamlit web application that performs real-time object detection and tracking using Artificial Intelligence and a webcam. The system uses the YOLOv8 model to detect different objects in live video streams.

## Features
- Real-time webcam detection
- Object tracking using YOLOv8
- Object counting
- Alert system for specific objects
- Save detected frames as images
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- OpenCV
- YOLOv8
- Ultralytics

## Project Files
- `app.py` – Main Streamlit application
- `yolov8n.pt` – YOLOv8 trained model
